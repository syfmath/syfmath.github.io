Ito Formula, Ornstein–Uhlenbeck Process

Yifan Shi
October 17, 2024

1. Stochastic Calculus

If we begin from constructing stochastic integrals, the content becomes extensive and somewhat unnecessary. Thus, only some intriguing aspects are highlighted here:

  1. The result of a stochastic integral depends on the choice of the sampling point in partitioning the interval. Commonly, the left endpoint is chosen, though other methods also find applications in various fields.
  2. There exists a one-to-one correspondence between the space of all uniformly bounded martingales \begin{cases} H^2(\Omega, \mathcal{F}, F, P) \end{cases} and the random variable space \begin{cases} L^2(\Omega, \mathcal{F}, P) \end{cases}.

2. Ito Formula

The Ito formula is essential in stochastic calculus, serving as the chain rule for derivatives. Here, we present a specific version used for the Ornstein–Uhlenbeck process.

2.1 Related Definitions and Lemmas

  • Def 1: Simple Progressive Measurable Process

    H_s(\omega) = \sum_{i=0}^{n-1} \xi_i(\omega) \mathbb{1}{[t_i, t{i+1})}(s)

    where \begin{cases} 0 = t_0 < t_1 < \dots < t_n \end{cases} and \begin{cases} \xi_i \end{cases} is a bounded \begin{cases} \mathcal{F}_{t_i} \end{cases}-measurable random variable.

  • Def 2: Progressive Measurable Process

    A process \begin{cases} X : [0, t] \times \Omega \to \mathbb{R} \end{cases} is measurable with respect to \begin{cases} \sigma \end{cases}-field \begin{cases} B([0, t] \times \mathcal{F}_t) \end{cases}.

  • Lemma 1: The set \begin{cases} E \end{cases} of all simple progressive measurable processes is dense in \begin{cases} L = { M(t) : \mathbb{E} \int_0^T M^2(u) du < +\infty, M_0 = 0 } \end{cases}.

  • Lemma 2: (Ito Isometry) For any progressive measurable process \begin{cases} X(t) \end{cases}:

    \begin{cases}
    \mathbb{E} \left( \int_0^t X(u) dW(u) \right)^2 = \mathbb{E} \int_0^t X^2(u) du = \int_0^t \mathbb{E} X^2(u) du
    \end{cases}

2.2 Ito Formula

Understanding Ito’s formula can be simplified using three principles:

  1. \begin{cases} dW_t dW_t = dt \end{cases}
  2. \begin{cases} dW_t dt = 0 \end{cases}
  3. \begin{cases} dt dt = 0 \end{cases}

Theorem 1 (Quadratic Variation)

For \begin{cases} g(x) \in C(\mathbb{R}) \end{cases} with \begin{cases} \int_0^T \mathbb{E}[g^2(W(u))] < +\infty \end{cases}, and \begin{cases} W(t) \end{cases} being a Wiener process:

\begin{cases}
\lim_{|\pi_n| \to 0} \sum_{i=0}^{n-1} g(\theta_i) |W(t_{i+1}) - W(t_i)|^2 = \int_0^t g(W(u)) du
\end{cases}

Theorem 2 (Ito Formula)

Assume an Ito process \begin{cases} dX_t = \mu_t dt + \sigma_t dW_t \end{cases}. For \begin{cases} f(x, t) \in C^2 \end{cases}, we have:

\begin{cases}
df(X_t, t) = \left( \frac{\partial f}{\partial t} + \frac{\partial f}{\partial X_t} \mu_t + \frac{1}{2} \frac{\partial^2 f}{\partial X_t^2} \sigma_t^2 \right) dt + \frac{\partial f}{\partial X_t} \sigma_t dW_t
\end{cases}

3. Construction of Ornstein–Uhlenbeck Process

Introduction

A continuous, centered Gaussian process \begin{cases} X \end{cases} on \begin{cases} \mathbb{R} \end{cases} with covariance function \begin{cases} r_{s,t} = e^{|t-s|} \end{cases} is known as a stationary Ornstein–Uhlenbeck process. This process is expressed as \begin{cases} X_t = e^{t}B(e^{2t}) \end{cases} in terms of Brownian motion \begin{cases} B \end{cases}.

The classical SDE for the Ornstein–Uhlenbeck process:

\begin{cases}
dX_t = \theta(\mu - X_t) dt + \sigma dW_t
\end{cases}

where:

  • \begin{cases} \theta > 0 \end{cases}: speed of mean reversion,
  • \begin{cases} \mu \end{cases}: mean,
  • \begin{cases} \sigma > 0 \end{cases}: volatility,
  • \begin{cases} W_t \end{cases}: Wiener process.

3.1 Solution

Using \begin{cases} f(X_t, t) = X_t e^{\theta t} \end{cases} and applying Ito’s Lemma:

\begin{cases}
df(X_t, t) = \left( X_t \theta e^{\theta t} + e^{\theta t} \theta (\mu - X_t) \right) dt + e^{\theta t} \sigma dW_t = \theta \mu e^{\theta t} dt + e^{\theta t} \sigma dW_t
\end{cases}

Integrating both sides from 0 to \begin{cases} t \end{cases}:

\begin{cases}
X_t = (1 - e^{-\theta t}) \mu + X_0 e^{-\theta t} + \sigma \int_0^t e^{-\theta (t-s)} dW_s
\end{cases}

with variance:

\begin{cases}
\text{Var}\left(\sigma \int_0^t e^{-\theta (t-s)} dW_s \right) = \frac{\sigma^2 (1 - e^{-2\theta t})}{2\theta}
\end{cases}

3.2 Properties

  • Expectation: For \begin{cases} X_t = u \end{cases}:

    \begin{cases}
    \mathbb{E}(X_{t+\tau} | X_t = u) = (u - \mu) e^{-\theta \tau} + \mu \to \mu \quad \text{as } \tau \to \infty
    \end{cases}

  • Variance: For \begin{cases} X_t = u \end{cases}:

    \begin{cases}
    \text{Var}(X_{t+\tau} | X_t = u) = \frac{\sigma^2 (1 - e^{-2\theta \tau})}{2\theta} \to \frac{\sigma^2}{2\theta} \quad \text{as } \tau \to \infty
    \end{cases}

  • Covariance:

    \begin{cases}
    \text{Cov}(X_s, X_t) = \frac{\sigma^2}{2\theta} \left( e^{-\theta |s - t|} - e^{-\theta (s + t)} \right)
    \end{cases}

End of Notes
